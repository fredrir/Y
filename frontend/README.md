# Y (Twitter klone)

Stakken vi har valgt er: React Typescript og Vite. Vi har valgt tailwindcss og shadcn/ui for styling og komponenter.

For å kjøre prosjektet, sørg for at du har node installert. Videre kan du kjøre

```
npm install
```

og

```
npm run dev
```
